# APPF Frontend

Frontend React para APPF.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Configuração

- O proxy de desenvolvimento está configurado para `http://127.0.0.1:8000`.
- A API chama rotas como `/auth/token`, `/dados/importar`, `/dados/importar/preview-detalhado`.
